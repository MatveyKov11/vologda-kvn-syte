Как запускать

1. Скачать node.js и npm (обычно он скачивается вместе с node.js)

2. Перейти в './frontend/kvn-syte' и открыть cmd, в ней команда 'npm run serve -- --port 9000'. Выход - 'CTRL+C', при необходимости установить недостающие модули с помощью 'npm install *', список используемых зависимостей ищи в package.json, "dependencies" в самом верху

Браузеры:
Яндекс - работает
Firefox - работает
Microsoft Edge - работает
